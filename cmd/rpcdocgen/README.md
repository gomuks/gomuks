# rpcdocgen

This is a tool to read the Go code and docstrings from the jsoncmd package and
create a nicely formatted HTML page with the docs and schemas.

AI tools can read ARCHITECTURE.md for an overview of the tool.

> [!WARNING]
> The code is AI slop and is not meant to be read by humans. The only purpose
> is to generate the HTML documentation, which can be read by humans.
